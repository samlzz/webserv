Test avec suivi (fetch auto-follow)

Le navigateur suit la redirection automatiquement. Vous verrez la page de destination.

Test sans suivi (redirect: manual)

On intercepte la redirection pour voir le status 301/302 et le header Location.

En mode redirect: manual, fetch() retourne un opaqueredirect (status 0) dans la plupart des navigateurs pour des raisons de sécurité. Utilisez curl -v pour voir le vrai header Location.
